# Mining Simulation

Folder ini berisi simulasi proof-of-work untuk belajar konsep mining.

Ini bukan miner crypto beneran, tidak terhubung ke wallet, pool, RPC, atau jaringan blockchain mana pun.

## Cara menjalankan

```powershell
python toy_miner.py
```

## Ubah tingkat kesulitan

Naikkan nilai `DIFFICULTY` di `toy_miner.py`.

Semakin besar angkanya, semakin lama proses mencari hash yang cocok.
